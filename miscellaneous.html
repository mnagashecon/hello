R code to calculate summary statistics (N, mean, sd) and output a LaTeX tabular.
I hate it when people say, use package A and code this, use package B, etc. when they don't actually write the package. You spend days and nights to figure out how to use the functions in it. In several years the package will be abandoned and your results will be irreproducible. Better not to rely too much on others' works. Particularly when I have some routine works that I wanna do in my favourite way.
